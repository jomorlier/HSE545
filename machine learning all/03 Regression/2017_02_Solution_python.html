Problem 1¶

The regularized least-squares problem has the form


$$ \min_{\theta} \;\lVert A\theta -y\rVert_2^2 + \lambda \lVert \theta \rVert_2^2$$

  1. Show that the solution is given by

    $$ \hat{\theta} = \left( A^T A + \lambda I_n \right)^{-1} A^T y $$ (Do not use the method of Lagrangian multipliers)

  2. Find and draw an approximated curve of the given data points in Python
    • overcome overfitting
    • use 10 RBFs with $\sigma = 1$
    • choose a proper value of $\lambda$ on your own

Solution¶

$$ \begin{align*} \left\Vert A\theta - y\big\Vert^2_{2} + \big\Vert \sqrt{\lambda}I\theta-0\right\Vert^2_{2} & = \Bigg\Vert \begin{bmatrix} A\\ \sqrt{\lambda}I \end{bmatrix} \theta- \begin{bmatrix} y\\ 0 \end{bmatrix} \Bigg\Vert^2_{2} \end{align*}$$$$ \hat{\theta} = \left( \begin{bmatrix} A\\ \sqrt{\lambda}I \end{bmatrix}^T \begin{bmatrix} A\\ \sqrt{\lambda}I \end{bmatrix}\right)^{-1}\begin{bmatrix} A\\ \sqrt{\lambda}I \end{bmatrix}^T \begin{bmatrix} y\\ 0 \end{bmatrix} = \left( A^T A + \lambda I_n \right)^{-1} A^T y $$

import numpy as np
import matplotlib.pyplot as plt
from numpy.linalg import inv

x = np.linspace(-4.5, 4.5, 10).reshape(-1, 1)
y = np.array([0.9819, 0.7973, 1.9737, 0.1838, 1.3180, -0.8361, -0.6591, -2.4701, -2.8122, -6.2512]).reshape(-1, 1)

xp = np.arange(-4.5, 4.5, 0.01).reshape(-1, 1)

d = 10

u = np.linspace(-4.5, 4.5, d)
sigma = 1

A = np.hstack([np.exp(-(x-u[i])**2/(2*sigma**2)) for i in range(d)])
rbfbasis = np.hstack([np.exp(-(xp-u[i])**2/(2*sigma**2)) for i in range(d)])

A = np.asmatrix(A)
lamb = 0.1

theta = inv(A.T*A + lamb*np.eye(d)*A.T)*A.T*y
yp = rbfbasis*theta

plt.figure(figsize=(10, 6))
plt.plot(x, y, 'o')
plt.plot(xp, yp)
plt.grid(alpha=0.3)
plt.axis([-5, 5, -12, 6])
plt.show()

Problem 3¶

If $X$ is on the linf of $\omega^TX + \omega_0 = 0$, the objective function can be interpreted as the shortest distance $r^{*}$ from origin to the line. We geometrically (or intuitively) know that distance $r$ will be minimal when vector $X$ is orthogonal to the line


$$r^{*} = -\frac{\omega_0}{\lVert \omega \rVert} \text{and} \vec X^* = -\frac{\omega_0}{\lVert \omega \rVert}\frac{\vec \omega}{\lVert \omega \rVert} = \frac{\omega_0}{\omega^T\omega}\vec \omega$$

import matplotlib.pyplot as plt

from six.moves import cPickle

data = cPickle.load(open('./data.pkl', 'rb'))

data0 = data['0']
data1 = data['1']

# Convert to binary image

data1 = data1 > 125

# Average pixels locaed at the center of the image
feature0_1 = np.sum(data0[:,10:20,10:20], axis=(1, 2)).reshape(-1, 1)/100
