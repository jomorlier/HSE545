0. Supervised learning¶

  • Given training set $\left\{ \left(x^{(1)}, y^{(1)}\right), \left(x^{(2)}, y^{(2)}\right),\cdots,\left(x^{(m)}, y^{(m)}\right) \right\}$
  • Want to find a function $g_{\theta}$ with learning parameter, $\theta$
    • $g_{\theta}$ desired to be as close as possible to $y$ for futer $(x,y)$
    • $i.e., g_{\theta}(x) \sim y$
  • Define a loss function $\ell$
  • Solve the following optimization problem:
$$ \begin{align*} \text{minimize} &\quad f(\theta) = \frac{1}{m} \sum_{i=1}^{m} \ell \left(g_{\theta} \left(x^{(i)}\right), y^{(i)}\right)\\ \text{subject to} &\quad \theta \in \Theta \end{align*} $$

In [132]:
N = 250 # number of points per class
D = 2 # dimensionality
K = 3 # number of classes
X = np.zeros((N*K,D)) # data matrix (each row = single example)
y = np.zeros(N*K) # class labels
for j in range(K):
    ix = range(N*j,N*(j+1))
    r = np.linspace(0.0,1,N) # radius
    t = np.linspace(j*4,(j+1)*4,N) + np.random.randn(N)*0.2 # theta
    X[ix] = np.c_[r*np.sin(t), r*np.cos(t)]
    y[ix] = j
