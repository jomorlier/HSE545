3. Training Neural Networks¶

Cost function

  • 우리의 모델이 예측하는 값과 실제 값이 얼마나 일치하는지 확인할 수 있는 function
$$ \min_{\theta} \sum_{i=1}^{m}\ell\left( h_{\theta}\left(x^{(i)}\right),y^{(i)}\right)$$

Backpropagation

  • Cost function이 최소가 되도록 전체 파라미터가 가야할 방향 (Gradient) 를 찾음

Gradient descent

  • 각각의 파라미터에 대해 Cost function이 최소가 되는 방향 (Gradient) 으로 파라미터를 업데이트
$$ \theta:= \theta - \alpha \nabla_{\theta} \left( h_{\theta} \left(x^{(i)}\right),y^{(i)}\right)$$

4.5. Define Network Structure¶

  • Network를 구성하는 구성요소와 TensorFlow code 설명
  • 일반적으로 Network는 Linear combination, Nonlinear activation, Classification (Fully connected) 로 이루어져 있음

1) Linear combination

$$y_j = \left(\sum\limits_i \omega_{ij}x_i\right) + b_j$$$$\mathcal{y} = \mathcal{w}^T \mathcal{x} + \mathcal{b}$$ 
layer1 = tf.matmul(x, weights['h1_w']) + biases['h1_b']

2) Nonlinear activation function

layer1 = tf.nn.relu(layer1)

3) Classification

output_layer = tf.matmul(layer1, weights['output_w']) + biases['output_b']


In [9]:
# Define Network
def net(x, weights, biases):
    # first hidden layer
    layer1 = tf.matmul(x, weights['h1_w']) + biases['h1_b']
    # non linear activate function
    layer1 = tf.nn.relu(layer1)
    
    # Output layer with linear activation
    output_layer = tf.matmul(layer1, weights['output_w']) + biases['output_b']
    return output_layer

4.6. Define Cost¶

  • Neural Net의 output과 label $y$ 의 차이가 최소가 될 수 있도록 cost를 정의
In [10]:
# Define Cost
learning_rate = 0.1
x = tf.placeholder(tf.float32, [None, n_input])
y = tf.placeholder(tf.float32, [None, n_classes])

nn_output = net(x, weights, biases)
cost = tf.reduce_mean(tf.square(nn_output - y))
optimizer = tf.train.GradientDescentOptimizer(learning_rate).minimize(cost)

    sess.run(optimizer, feed_dict={x: batch_x,  y: batch_y})
    c = sess.run(cost, feed_dict={x: batch_x,  y: batch_y})
    if epoch % flag == 0:
        print ("Iter : {}".format(epoch))
        print ("Cost : {}".format(c))
